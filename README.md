# Social-Media-Application
A social media application, named 'Capture'. I created Capture using Node.js for the backend, React Native for the frontend and MongoDB for the database. This application is compatible with multiple platforms, such as desktop, laptop, IOS devices, and Andriod devices. Users are allowed to post, comment, like, direct message, and follow each other.
